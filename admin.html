الاتصال بـ GitHub

لوحة التحكم تتصل مباشرة بمستودع الموقع على GitHub لحفظ كل التعديلات. محتاجة تدخلي Personal Access Token مرة واحدة فقط — يُحفظ في متصفحك فقط:

  1. افتحي صفحة github.com/settings/tokens?type=beta من حسابك على GitHub.
  2. اضغطي "Generate new token"، واختاري Repository access ← Only select repositories ← caroleehab.
  3. من Permissions، فعّلي صلاحية Contents: Read and write فقط، واتركي باقي الصلاحيات بدون تفعيل.
  4. اضغطي Generate token، انسخي التوكن، وضعيه في الحقل بالأسفل.

ملاحظة: التوكن يُحفظ على جهازك فقط (داخل المتصفح) ولا يُرسل لأي مكان غير GitHub. لا تشاركي هذا التوكن مع أي شخص آخر.
